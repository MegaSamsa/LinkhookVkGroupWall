LinkhookVk [Version 1.1]
------------------------------------------------------

Скрипт осуществляет сбор ссылок на новые посты в заданной группе ВК. Собранные ссылки записываются в Excel таблицу. Предусмотрена работа с закреплёнными постами (скрипт пропускает закреплённый пост). Закрытые группы просматривать нельзя :D

Структура проекта:
LinkhookVkGroupWall | Корневая папка проекта
    .gitignore      | Игнорирование публикации config.json и Ссылки.xlsx в репозиторий
    config.json     | Файл конфигурации
    init.py         | Файл с импортом библиотек
    Ссылки.xlsx     | Таблица с собранными ссылками
    main.py         | Главный исполняемый файл
    readme.txt      | Информация о проекте в формате txt
    run.bat         | Пакетный файл для быстрого запуска

Файл конфигурации config.json имеет параметры:
    group_id        | str   |   id ВК группы
    access_token    | str   |   токен доступа
    version         | str   |   версия API
    check_interval  | int   |   интервал проверок
    file_path       | str   |   путь к Excel файлу

Заполнение config.json:
    Получение id группы (group_id): перейти в нужную группу и скопировать id. Если у группы есть маска id, можно скопировать ссылку на любой пост в группе и оттуда достать id;
    Получение access токена (access_token): создать мини-приложение ВК типа "Мини-приложение" или "Плагин для сообществ". Перейти в раздел "Разработка" -> "Ключи доступа". Скопировать сгенерированный СЕРВИСНЫЙ ключ (официальная документация: https://dev.vk.com/ru/method/wall.getById);
    version = "5.131";
    check_interval = 60 (в секундах);
    file_path = путь к сохранению файла, например: "C:/Users/Admin/Desktop/".

При разработке использовался язык Python версии 3.10.5
